###.A.Азбука
```
x = int(input())
resultat = "YES"
for i in range(x):
    word = input()
    if word[0] not in ("а", "б", "в"):
        resultat = "NO"
        break
print(resultat)
```

### C.Анонс новости
```
length = int(input())
for _ in range(int(input())):
    line = input()
    print(line[: length - 3].ljust(length, ".") if len(line) > length else line)

```
