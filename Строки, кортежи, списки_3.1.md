###.A.Азбука
```
x = int(input())
resultat = "YES"
for i in range(x):
    word = input()
    if word[0] not in ("а", "б", "в"):
        resultat = "NO"
        break
print(resultat)
```

### C.Анонс новости
```
length = int(input())
for _ in range(int(input())):
    line = input()
    print(line[: length - 3].ljust(length, ".") if len(line) > length else line)

```
### D.Очистка данных
```
while (n := input()):
    if not n.endswith('@@@'):
        if n.startswith('##'):
            print(n[2:])
        else:
            print(n)    
```

### H.Зайка — 7
```
results = []  # Liste pour stocker les résultats

for _ in range(int(input())):
    n = input()
    if "зайка" in n:
        results.append(n.index("зайка") + 1)
    else:
        results.append("Заек нет =(")

# Afficher tous les résultats à la fin
for result in results:
    print(result)

```
